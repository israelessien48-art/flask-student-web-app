# Flask Student Web App

## Description
This is a Student Management Web Application created using Flask and SQLite.

The application allows users to:
- Add students
- View student records
- Search students
- Delete students

## Technologies Used
- Python
- Flask
- SQLite
- HTML

## How to Run
1. Install Flask:
pip install flask

2. Run the application:
python app.py

3. Open browser:
http://127.0.0.1:5000
